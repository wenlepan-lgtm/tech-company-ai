## 📋 常用命令

| 命令 | 说明 |
|------|------|
| `/团队` 或 `/roles` | 显示这个团队列表 |
| `/项目` 或 `/projects` | 查看所有项目 |
| `/进度` | 查看当前任务进度 |
| `/新建项目 名称` | 创建新项目 |

## 💡 使用示例

```

@小研 实现JWT认证功能
@小审 审核刚才的代码
@小运 部署到测试环境
```
