## 项目目标
实现一个支持多端登录的用户认证系统,包括:
- 用户注册和登录
- JWT token 认证
- 多设备同时在线
- 登出和 token 刷新

## 项目范围
### 包含
- 用户注册功能(用户名/密码)
- 用户登录功能
- JWT token 生成和验证
- Token 刷新机制
- 多设备登录支持
- 登出功能
- 基本的单元测试

### 不包含
- 第三方登录(Google/GitHub等)
- 双因素认证(2FA)
- 密码找回功能
- 用户资料管理
- 权限管理系统

## 技术栈
- **后端**: Python 3.11
- **框架**: FastAPI
- **认证**: JWT (PyJWT库)
- **缓存**: Redis (会话存储)
- **数据库**: PostgreSQL
- **测试**: pytest

## 关键约束
- **时间约束**: 1周内完成核心功能
- **资源约束**: 单人开发
- **技术约束**: 必须使用现有技术栈,不引入新框架
- **安全约束**: 密码必须加密存储,token 有效期不超过24小时
