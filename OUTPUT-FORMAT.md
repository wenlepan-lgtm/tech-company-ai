# 输出格式规范

> 所有角色输出必须遵循此格式

---

## 通用格式

每个角色完成任务后，必须输出：

```
【任务反馈】
━━━━━━━━━━━━━━━━━━━━━━
⏰ 时间: YYYY-MM-DD HH:MM:SS
👤 执行人: [角色名]
🛠️ 工具: [使用的工具]
📊 状态: [完成/进行中/失败]
━━━━━━━━━━━━━━━━━━━━━━

[具体内容]
```

## 文档输出规则

**重要**：所有 .md 文件必须输出到飞书文档，而不是本地文件！

### 输出流程

1. 生成 Markdown 内容
2. 使用 `feishu-create-doc` 创建飞书文档
3. 返回文档链接给老板

### 示例

```
【文档已创建】
📄 标题: 项目架构设计
🔗 链接: https://www.feishu.cn/docx/doxcnXXXXXXXXXXXXXXXXXXX
```

---

## 时间格式

- **标准格式**: `YYYY-MM-DD HH:MM:SS`
- **时区**: GMT+8（北京时间）
- **精度**: 精确到秒

示例: `2026-03-14 22:40:15`
